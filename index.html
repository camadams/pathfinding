Bio
From my research, a popular 'algorithm project' on software developers' portfolios, was path finding. Most projects I saw used a grid and a feature to draw obstacles for the algorithm to find a path around. I wanted to try use a graph, which ended up taking most of my time. I used Djisktras algorithm to find the shortest path from any node to any other node.

Start by clicking a 'start' node, then clicking 'Generate Shortest Paths', then clicking any other node to visualise the shortest path.
